# 🤖 AI Cheat Sheets

Repositorio público de recursos, plantillas y guías para estructurar el uso de inteligencia artificial en contextos profesionales.

El objetivo es pasar de un uso improvisado de IA a un uso **estructurado, consistente y orientado a resultados**.

---

## 🎯 Propósito

Este repositorio busca:

- Estandarizar la forma en que se interactúa con modelos de IA  
- Mejorar la calidad y consistencia de los resultados  
- Reducir ambigüedad en los prompts  
- Facilitar la reutilización de estructuras probadas  
- Escalar el uso de IA en distintos contextos de negocio  

---

## 🧱 Estado del proyecto

🚧 **En construcción**

El repositorio se está desarrollando por módulos (capítulos).  
Cada módulo aborda un tipo específico de uso de IA.

---

## 🧠 Módulo 1: Prompt Generator

Primer módulo funcional del repositorio.

Enfocado en un problema clave:

> **Cómo diseñar prompts de alta calidad que generen resultados consistentes.**

Este módulo introduce un enfoque estructurado para la creación de prompts, evitando dependencias en estilo personal o interpretación subjetiva.

---

## 📩 Submódulo: Emails

Este submódulo contiene plantillas diseñadas para la generación de prompts orientados a correos corporativos.

Las plantillas siguen principios comunes:

- Definición clara del contexto  
- Declaración explícita del objetivo  
- Separación de temas cuando aplica  
- Enfoque en acciones y resultados  
- Control del tono y nivel de formalidad  
- Estructura de salida consistente  

Plantillas disponibles:

- **Plantilla Full** → Para escenarios complejos o sensibles  
- **Plantilla Lite** → Para uso rápido y casos simples  

> Cada plantilla incluye su propia documentación detallada de uso.

---

## 🗂️ Versionamiento

Se mantiene historial de versiones anteriores de las plantillas para:

- Trazabilidad  
- Comparación de cambios  
- Evolución del diseño  

---

## ⚙️ Cómo usar este repositorio

1. Selecciona el módulo correspondiente  
2. Elige la plantilla adecuada según tu caso  
3. Diligencia los campos indicados en la plantilla  
4. Ejecuta el prompt en tu herramienta de IA  
5. Valida el resultado generado  

Regla práctica:

> Complejidad alta → usa plantilla Full  
> Uso rápido → usa plantilla Lite
